# pixie-event-tracker
Event discovery and tracking tool using Python
Event Discovery & Tracking Tool

This project extracts public event data from BookMyShow for a selected city
and stores it in Google Sheets.

Features:
- City-based event extraction
- Deduplication using event URLs
- Expired event marking
- Designed for scheduled execution using cron

Tech Stack:
- Python
- BeautifulSoup
- Google Sheets API

How to Run:
1. Install dependencies from requirements.txt
2. Run event_tracker.py
